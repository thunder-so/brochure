## What is Thunder?

Thunder is a cloud deployment platform that enables you to effortlessly deploy your applications to AWS with minimal configuration. Whether you're working with static sites, single-page applications, or complex web apps, Thunder simplifies the deployment pipeline, allowing you to focus on building great products rather than managing infrastructure.

## Key Features

- **One-Click Deployments**: Import your repositories and deploy to AWS with just a few clicks.
- **Built-in CI/CD Pipelines**: Automated builds and deployments triggered by your Git commits.
- **Custom Domain Support**: Easily configure custom domains and SSL certificates.
- **Global CDN Distribution**: Leverage Amazon CloudFront for fast, global content delivery.
- **Environment Variables Management**: Securely manage your environment variables with AWS Parameter Store.
- **Scalable Infrastructure**: Automatically scalable infrastructure built on AWS best practices.
