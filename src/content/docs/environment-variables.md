---
title: 'Environment variables'
description: 'Understand how to configure and manage environments in Thunder, including AWS account and region selection, environment variables, and secrets management.'
---

Thunder uses AWS Parameter Store to manage environment variables on your account.

- AWS Parameter Store integration: Thunder securely stores your environment variables.
- Dynamic configuration: Set environment-specific variables for your application.
- Build-Time usage: Variables are accessed during the build step process in your deployment pipeline.
